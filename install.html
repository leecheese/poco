워드프레스 설치하기💪

설치형? 호스팅? 이게 다 무슨 말인지 궁금하신 분들 계셨다면 푸쳐핸섭✋
5분만에 끝내는 워드프레스, 첫 시간은 설치로 시작해볼까요?

#세팅법 #워린이 #무료로_시작하기 #닷홈

워드프레스는 CMS라고 하는, 웹사이트와 대시보드를 모두 커스텀할 수 있는 제작 시스템을 말합니다. 윅스, 아임웹들과는 다르게 호스팅을 직접 해 설치할 수 있고 웹사이트를 0부터 내 마음대로 꾸밀 수 있죠. 기존의 웹 빌더와는 다른 워드프레스, 한번 설치하는 시간을 가져볼까요?

크게 다음과 같은 목차로 진행하게 됩니다.

워드프레스 설치 순서 핵심 정리

  • 1.
    닷홈
    에서 무료 호스팅하기 (광고 아님)
  • 2. 워드프레스 내부에 설치하기
  • 3. 워드프레스에서 관리자 계정으로 로그인하기
  • 4. 테마 및 플러그인 설치하기

    • 이번 게시글에서는 전체 순서 중 1에서 3까지만 진행하게 될 예정입니다.
    설치 과정은 아주 쉽습니다. 지금 바로 시작합시다.

    닷홈
    에서 무료 호스팅하기✍️

    👀 광고가 아닌 실제 체험 후 남기는 글이며, 연습용으로 만드는 분들을 대상으로 합니다.
    유료 호스팅과 홈페이지에 대해서는 다른 글로 찾아올 예정이니 조금만 기다려주세요!

    닷홈에 접속하여 회원가입을 따로 진행합니다. (추천인이나 광고 일체 없음!)
    회원가입 이후에는 상단 메뉴에서 웹호스팅을 클릭하여 무료 호스팅을 클릭합니다.

    무료 호스팅에 대한 안내가 보여집니다. 신청하기를 클릭합니다.
    무료 호스팅으로 만든 사이트는 약 3개월 동안 유지되며, 이후에는 폐기됩니다.

    CMS 가입 정보 입력하기✍️

    신청하기 버튼을 누른 이후 표시되는 정보를 입력합니다.
    여러 정보를 오타 없이 적어주되 가장 중요한 내용을 입력해야 하는데요.
    바로 CMS 가입 정보입니다. CMS 자동 설치 항목에서 나오는 워드프레스 한글을 꼭 선택해주세요!

    CMS 계정과 FTP 계정을 헷갈리는 경우가 많아, 비밀번호는 동일하게 입력해주셔도 좋습니다.

    완료되면 다음처럼 화면이 뜨며 워드프레스에 로그인할 수 있게 됩니다.

    워드프레스 접속하기🙌

    내 주소로 접속하는 가장 빠른 방법이 있는데요, 주소창에 다음과 같이 입력해줍니다.

    [가입했던 FTP 아이디].dothome.co.kr/wp-admin

    주소창에 검색하면 다음과 같이 페이지가 이동하는데요, 앞에서 가입했던 CMS 계정의 아이디와 비밀번호를 입력해줍니다.

    성공적으로 로그인하면, 워드프레스 초기 세팅 화면에 진입할 수 있습니다.

    좋아요, 아주 잘하셨어요! 다음 강의로 넘어가볼까요?
